# Marauder-Zero
Marauder-Zero regroupe le développement et la documentation pour porter/adapter le firmware Marauder sur un Flipper Zero, en utilisant un M5 Stack Tab5 configuré comme point d’accès de laboratoire. Le projet a un objectif pédagogique : démonstration et évaluation contrôlée de vulnérabilités Wi-Fi uniquement sur des réseaux de test dûment autorisés.
